    구글 드라이브 연동 설정

    구글 API 정책에 따라 데스크톱 앱에서 개인 구글 드라이브에 직접 연결하려면 클라이언트 인증키 (client_secrets.json) 파일이 필요합니다.

    🔑 인증키 발급 및 등록 순서
    1. 구글 클라우드 콘솔 방문 (로그인 후 프로젝트 생성)
    2. 사용자 인증 정보 만들기 > OAuth 클라이언트 ID를 클릭합니다.
    3. 애플리케이션 유형을 데스크톱 앱 (Desktop app)으로 선택하고 만듭니다.
    4. 생성된 클라이언트 오른쪽의 다운로드 버튼을 눌러 JSON 파일을 받습니다.

    받으신 JSON 파일을 아래 버튼을 클릭해 선택하면 자동으로 프로그램에 등록됩니다.
