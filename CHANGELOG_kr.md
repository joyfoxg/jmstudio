### 🚀 v3.9.28 (2026-06-14) - 캔버스 다중 선택, 그룹화 및 단축키 기능 전면 구현 [최신안정]
*   **옵시디언 호환 인피니트 캔버스 고도화**:
    *   **다중/영역 선택(Multi/Lasso Select)**: 캔버스 내 마우스 영역 지정 및 다중 개체 선택 인터랙션 추가.
    *   **그룹화(Grouping)**: 선택된 노드들을 묶어주는 시각적 그룹 컨테이너(Group Node) 생성 기능 완비.
    *   **빠른 복제(Duplicate)**: 캔버스 상의 노드들을 원클릭 혹은 단축키로 즉시 복제하는 기능 탑재.
    *   **마크다운 편집 전용 단축키**: 캔버스 내 마크다운 카드 타이핑 시 특화된 키보드 단축키 체계 구축.

### 🚀 v3.9.27 (2026-06-03) - 내 미디어 아코디언 분리, 전용 아이콘, 미리보기 및 저장 잠금 탑재 [안정]
*   **사이드바 내 미디어(My Media) 아코디언 분리**:
    *   내서재 목록에서 그림 및 미디어 파일들(images, PDFs, audio, video)을 감추고, 새로 신설된 `내 미디어` 아코디언 패널에 별도 분리하여 노출하도록 트리 구조 스플리팅 로직을 도입했습니다.
    *   문서와 미디어가 동일 폴더에 있더라도 각각의 영역에 구조가 완전하게 보존되어 렌더링됩니다.
*   **미디어 파일 종류별 전용 아이콘**:
    *   그림파일(`image`), 동영상(`video`), 오디오(`music`) 등 파일 확장자를 자동 분석하여 파일 트리 탐색기 내에 각각 어울리는 아이콘을 동적으로 매핑했습니다.
*   **미디어 미리보기 전용 뷰어 및 에디터 비활성화**:
    *   미디어 파일 클릭 시 에디터 영역이 흐려지며 읽기 전용 상태가 되고, 우측 라이브 미리보기창에 해당 파일의 전용 플레이어/뷰어(이미지 뷰어, PDF 인라인 프레임, 오디오/비디오 HTML5 플레이어 등)가 자동으로 로드됩니다.
    *   외부 경로 파일에 주소 접두사(`/workspace/`)가 중복 부착되어 404 및 엑박이 뜨던 버그를 정밀 해결했습니다.
*   **이진 파일 파괴 방지 저장 잠금**:
    *   에디터 상태에서 저장(`Ctrl+S` 또는 버튼 클릭)을 실행할 때 원본 미디어 파일 위에 빈 텍스트를 덮어씌워 바이너리를 손상시키는 일을 방지하도록 저장 시도를 원천 차단하고 경고 토스트를 노출합니다.

### 🚀 v3.9.26 (2026-06-03) - 위키 링크, 백링크 패널 및 Obsidian 호환 인피니트 캔버스 구현 [안정]
*   **Obsidian 100% 호환 인피니트 캔버스 (Infinite Canvas) 개발 완료**:
    *   D3-Zoom 기반 무한 패닝, 마우스 및 트랙패드 핀치 줌 배율 제어 및 16px 네온 격자 그리드 배경화면 연동.
    *   노드 간 베지어 연결선(SVG Edge) 실시간 드로잉 및 드롭 연동.
    *   마크다운(캐싱 렌더링), 이미지, PDF(Iframe) 카드 임베드 및 Lazy CodeMirror 6 편집기 동적 마운트.
    *   캔버스 내 무한 Undo/Redo 단축키(`Ctrl+Z`, `Ctrl+Y`) 및 즉시 저장(`Ctrl+S`) 단축키 연동.
    *   연결선(Edge) 선택 기능 및 HSL 색상 변경(6색) 및 삭제 플로팅 툴바 연동 (클릭 판정 범위를 28px로 대폭 확장하여 선택 성능 극대화).
    *   마우스 기본 커서를 윈도우 화살표 커서(`default`)로 교체 및 드래그 시 이동 커서(`move`) 연동.
    *   폴더 임베딩 카드(디렉토리 뷰어) 구현: 사이드바 폴더 드래그 앤 드롭 지원, 카드 내 Grid/List 전환, 파일명 실시간 필터링 검색, A-Z/Z-A 정렬, 새 마크다운 파일 생성(`+`), 부모 폴더 가기(`..`), 파일 클릭 시 캔버스 즉시 닫고 해당 파일 에디터로 전환 기능 완비.
    *   다른 이름으로 저장(Save As) 대화상자 연동: 윈도우 표준 저장 대화상자를 띄워 경로와 이름을 지정하여 `.canvas` 파일로 내보내기 지원.
*   **양방향 위키 링크 (`[[문서명]]`) 파싱 및 실시간 렌더링**:
    *   CodeMirror 6의 `ViewPlugin` 및 `WidgetType`을 활용하여, 비활성화된 편집 줄의 위키 링크 문법을 클릭 가능한 보라색 네온 단추로 자동 치환하는 플러그인을 탑재하였습니다.
    *   버튼 클릭 시 서재 내에서 대소문자 무관하게 해당 문서를 실시간 검색하여 열어주고, 문서가 없는 경우 워크스페이스 루트 아래에 YAML Front Matter(`tags: [위키링크]`)가 작성된 새 문서(`.md`)를 자동으로 생성하여 로드합니다.
*   **백엔드 참조 색인 실시간 연동 및 backlinks.json 색인화**:
    *   문서 저장(`save_file`) 시 본문 내 링크 관계를 파싱하여 `.jmstudio/backlinks.json` 색인을 실시간으로 업데이트하는 엔진을 완비했습니다.
    *   앱 최초 기동 시 비동기 스레드로 서재 전체 문서를 스캔하여 유실되거나 손상된 backlinks.json 파일을 자동 빌드/복원합니다.
*   **좌측/우측 듀얼 백링크 내비게이터 탑재**:
    *   **좌측 사이드바**: 파일 트리 탐색기 아래에 아코디언 형태로 현재 문서를 링크하고 있는 타 문서 목록을 노출합니다.
    *   **우측 미리보기**: 미리보기 하단에 프리미엄 백링크 카드 그리드를 렌더링하여 문서 간 입체적인 흐름 추적을 제공합니다.
*   **지식 그래프 (Force Graph) 문서 성격별 아이콘 시각화**:
    *   지식 그래프의 각 노드(문서)에 해당 문서의 성격(학술, 화학, 주식, 프로젝트, 일기, 일정, 위키 등 11개 카테고리)을 나타내는 이모지 아이콘을 표시하도록 시각화 엔진을 고도화했습니다.
    *   노드 중심에 이모지를 렌더링하고, 반투명 배경 원형 및 네온 글로우 외곽 링 효과를 가미하여 그래픽 뷰의 가독성과 미적 디자인을 완성했습니다.
    *   캔버스 2D 줌 인/아웃 배율에 비례하여 노드 크기와 이모지 폰트 크기가 자연스럽게 스케일링되도록 기하 정렬을 개선했습니다.
*   **되돌리기(Undo) / 다시실행(Redo) 런타임 오류 완벽 해결**:
    *   에디터 내에서 실행 취소 및 재실행 시 발생하던 `window.undoManager.undo is not a function` JS TypeError 오류를 해결했습니다.
    *   CodeMirror 6의 공식 명령 패키지(`@codemirror/commands`)의 `undo`와 `redo` 호출 방식으로 에디터 내 히스토리를 완벽하게 매핑하고, 하위 호환성을 위한 mock 객체 바인딩을 적용했습니다.
*   **프리미엄 네온 다크테마 CSS 최적화**:
    *   위키 링크 버튼, 사이드바 항목, 백링크 카드 단추에 대한 호버 인터랙션, 글로우 그림자 및 HSL 반응형 테마 스타일을 CSS 단독 규칙으로 통합 정의하여 인라인 CSS 코드를 걷어내고 렌더링 완성도를 극대화했습니다.

*   **드라이브 파일 스코프 안전 격리**: 구글 드라이브 전체 파일이 아닌, 본 애플리케이션 전용 생성 폴더(`JoyMarkdownStudio/`) 내의 자산에만 접근 권한(`drive.file`)을 인가하여 보안성을 원천 강화했습니다.
*   **양방향 동기화 및 시간차 파일 충돌 방지**: 로컬 저장 즉시 백그라운드 자동 클라우드 동기화를 지원하며, 로컬과 클라우드 파일의 최종 수정 시간(`mtime`) 차를 판독하여 충돌 시 스마트하게 선택하는 모달창을 탑재했습니다.
*   **원격 보관 조회 및 내려받기**: 클라우드 폴더를 트리뷰 하단 아코디언에서 직접 스캔하여, 현재 로컬 워크스페이스에 존재하지 않는 드라이브 내 문서를 원버튼으로 전격 Import할 수 있습니다.
